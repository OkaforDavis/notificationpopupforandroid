## 🎓 Example Workflow

```javascript
// 1. User clicks a button
document.getElementById('saveBtn').onclick = async () => {
    // 2. Show loading message
    Notification.info('Saving...', 'Please wait', 0);
    
    try {
        // 3. Make API call
        const response = await fetch('/api/save', {
            method: 'POST',
            body: JSON.stringify(data)
        });
        
        // 4. Clear loading and show success
        Notification.clearAll();
        Notification.success('Saved!', 'Your changes have been saved');
        
    } catch (error) {
        // 4. Or show error
        Notification.clearAll();
        Notification.error('Error!', 'Could not save changes');
    }
};
```

## ❓ Troubleshooting

**Q: Notifications not showing?**  
A: Make sure both `styles.css` and `notification.js` are included.

**Q: Wrong position?**  
A: Edit the CSS for `.notification-container` in `styles.css`.

**Q: Want different colors?**  
A: Edit the CSS variables at the top of `styles.css`.

**Q: Need help?**  
A: Check the relevant documentation file or open `examples.html`.
